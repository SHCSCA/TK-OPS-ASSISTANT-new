# v1.3.0 发布记录

发布日期：2026-03-30

## 变更摘要

- 视频剪辑模块接入真实持久化模型：新增视频工程、序列、片段、字幕、快照、导出记录等数据链路，基础剪辑不再依赖示例假数据。
- 接入视频编辑服务、Bridge 合同和 FFmpeg 导出运行时，支持素材挂载、片段裁切、字幕维护、工程快照与导出成片。
- `video-editor` / `visual-editor` 页面按模块拆分出独立 factory、loader、bindings，`page-loaders.js` 保留聚合入口，降低大文件继续膨胀的风险。
- 视频剪辑页改为 `static / timeline / outputs` 分层增量刷新，数据变更不再默认整页重渲染，减少加载抖动和重复绑定带来的卡顿。
- 修复 `video-editor-main.js` 语法损坏与运行时摘要接线问题，并让视频编辑迁移在旧库已预建部分视频表的情况下继续安全升级到 `3c7a6f5e9d21`。

## 重点改动

- 前端：
  - 新增 `desktop_app/assets/js/page-loaders/video-editor-main.js`

- 后端：

  - 接入视频编辑服务、导出服务和 Bridge 槽函数
  - 打包链纳入 FFmpeg 运行时资源
- 测试：
  - 新增视频剪辑运行时、bindings 拆分、加载器拆分与数据库生命周期回归测试
  - 补充视频剪辑语法校验与旧库升级兼容护栏
