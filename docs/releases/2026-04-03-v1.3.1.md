# v1.3.1 发布记录

发布日期：2026-04-03

## 变更摘要

- 新增发布前统一门禁脚本 `scripts/preflight-gate.ps1`，支持 `-Quick` 与 `-Full` 两档检查，并输出汇总结果与退出码。
- `release.ps1` 默认接入 `preflight-gate -Quick`，发布前自动执行关键回归检查，减少人工漏检风险。
- `scripts/dev.ps1` 启动链路修正：
  - 默认改为“仅启动 Tauri 桌面宿主，runtime 由宿主托管”
  - 去除脚本中文乱码输出
  - 浏览器调试模式下的外置 runtime 会在退出时自动回收
- 账号管理新增“CSV 文本导入向导”最小闭环：
  - `POST /accounts/import/preview`
  - `POST /accounts/import/apply`
  - 前端账号页支持预检查与执行导入（可选覆盖已存在账号）。
- 账号活动记录从“摘要展示”升级为“可查询”：
  - 支持关键词过滤（`query`）
  - 支持条数控制（20/50/100）
- 账号审计中心与生命周期规则收口：
  - `/accounts/{id}/activity` 新增 `category`、`severity` 筛选，并返回 `filters` 与活动级别字段。
  - 新增 `/accounts/{id}/lifecycle`，统一停用/归档/恢复/删除动作边界。
  - 前端账号页新增生命周期原因、二次确认、删除强确认（输入 `DELETE <username>`）与批量停用/恢复入口。
- 新桌面壳补齐全量菜单骨架：
  - 新增统一路由清单 `routeManifest.ts`
  - 左侧菜单改为从路由清单动态生成
  - 旧壳 `44` 个顶层页面已全部进入新壳菜单与占位路由，后续改为按菜单顺序逐页迁移
- 发布治理文档补齐：
  - `docs/releases/preflight-checklist.md`
  - `docs/releases/rollback-playbook.md`
